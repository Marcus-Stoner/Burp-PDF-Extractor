# Burp-PDF-Extractor

Simple PDF viewer extension for BurpSuite

## Features

PDF reader extension for BurpSuite, supporting the following features:
* Stores all unique PDF files into an artifacts dashboard
  *  Bulk save files
  *  Indivudally open files
* Open PDF files from requests using the hosts default PDF viewer
* Open PDF files from requests using a specific application
* Highlight all instances of PDFs in Burp Proxy
* Save PDF files

## Future Features
This was originally a vibe coded PoC for an extension that can be used for multiple file types. As Burp already supports previewing images, adding support for JPGs and PNGs is redundant so it has remained a PDF focused extension.

Any future ideas are welcomed.
